# tcalc

時間または時刻の計算機です。

## Usage

10時27分の42分後は次のように計算できます。

    $ tcalc 10:27 + 42
    11:09

時間（hours）と分（minuts）は `:` で区切って表現します。
数字だけなら分と解釈します。

演算子は `+` と `-` が使えます。
演算は2つ以上続けることもできます（常に前から処理します）。

9時から19時まで働いて休憩が1時間なら、実働時間は

    $ tcalc 19:00 - 9:00 - 1:00
    9:00

になります。
